## Getting Started

First, clone project:

```
git clone https://github.com/olgaAsmith/Harmony.git
```
Go to main directory: 
```
cd project-name
```
Then install dependencies:

```bash
npm install
#or
yarn install
```
Finally, run dev mode: 

```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
